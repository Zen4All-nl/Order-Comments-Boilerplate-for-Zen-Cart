This section identifies how the Order Comments Boilerplate, presents itself to the admin users.

This plug-in resolves the problem of repetitive order comments. Very often Orders require (almost) identical comments over and over again. Many shop owners keep a separate text file for storing these comments. Now you can store the comments in your database, and place them in the order comments field via a drop down menu.

Installation is pretty simple, as all but one files are new

  • Installation and Updates
  • Un-install

Notes:

There is one core-file change required by this plugin; you should always backup your cart's database and files prior to making any changes. Unzip the plugin's package file and then:

  1. Rename the YOUR_ADMIN folders to match your store's admin folder name:
  2. Copy the plugin's files (with the folders renamed as identified in step A) to your store's folder structure.
  3. Merge the core-file modifications required by this plugin with your store's existing copy of the file, using the plugin's version of the file if not currently present in your store's file-system. The code changes are clearly marked.
  4. Sign into your Zen Cart's admin panel.
