What does this mod do?

This section identifies how the Order Comments Boilerplate, presents itself to the admin users.

This plug-in resolves the problem of repetitive order comments. Very often Orders require (almost) identical comments over and over again. Many shop owners keep a separate text file for storing these comments. Now you can store the comments in your database, and place them in the order comments field via a drop down menu.

New install

Installation is pretty simple, as all but one files are new

Notes:

There is one core-file change required by this plugin; you should always backup your cart's database and files prior to making any changes. Unzip the plugin's package file and then:

  1. Rename the YOUR_ADMIN folders to match your store's admin folder name:
  2. Copy the plugin's files (with the folders renamed as identified in step A) to your store's folder structure.
  3. Merge the core-file modifications required by this plugin with your store's existing copy of the file, using the plugin's version of the file if not currently present in your store's file-system. The code changes are clearly marked like:
    /* BOF Zen4All Order Comment 1 of 3 */
    .......
    changed code block
    .......
    /* EOF Zen4All Order Comment 1 of 3 */    .
  4. Sign into your Zen Cart's admin panel.

Upgrade

When upgrading from a previous version, please use the flie list as a guide to see which file have been changed. Afte aplying the file changes, log in to your Zen Cart admin, to complete the installation. The database changes, if needed, will be executed automatically

Un-install

Use the file list to remove all the files, and changes made for this mod. Then log in to your Zen Cart Admin, and run the product_finder_uninstall.sql (found in the uninstall folder) in Admin=>Tools=>Install SQL Patches

Configuration

After installation, you will see a new menu item under tools

Click the link, and you will be presented with a new page. On this page you can create the boilerplate entries you want to use on the oreders page

Add/Edit comments

Editing a comments

After adding one ore more comments as a boilerplate, you can use them on the orders page.
When you are on the orders page editing an order, you will see a new dropdown just above the comments text area. From this dropdown you can select a predefined comment, which will automatically insertd into the commenst area. Be aware that all existing text inside de comments will be erased and replaced with the selected text. You can offcourse edit the text after selecting the boilerplate text.
